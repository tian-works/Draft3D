# User Guide

This page describes the typical Draft3D workflow in the GUI.

## Workflow

![Draft3D GUI workflow overview](../docs/assets/images/diagram.png)

1. **Sketch input**: draw or load a sketch.
2. **Generate images**: enter a prompt and generate candidates.
3. **Pick a result**: select your preferred image from the gallery.
4. **Generate 3D**: run the image-to-3D step and inspect the result in the 3D preview panel.

## Outputs

- Generated images and meshes are saved under `generated_images/` (organized by date).
